Icinga-Mobile
=============
[*English below...*](#introduction)

**Icinga-Mobile** là một ứng dụng cho android sử dụng API của icinga-web cho việc theo dõi, cảnh báo và cấu hình.

## Hướng dẫn nhanh

Download bản mới nhất từ trang github này.
